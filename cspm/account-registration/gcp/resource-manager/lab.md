## Overview

Register GCP projects (or an entire Organization) with CrowdStrike Falcon Cloud Security using Google Cloud Deployment Manager or gcloud CLI. Deploys the required Service Account, Workload Identity Federation, and Pub/Sub resources for IOA/IOM telemetry without requiring Terraform.
